About the Project:

This project is an assignment from The Odin Project's Full Stack JavaScript curriculum. The project involves creating a book library application that allows users to add and remove books, as well as update their reading status. The Odin Project is an online platform that provides free and open-source web development courses. The application involves creating a simple array to store book objects, writing a function to add books to the array, displaying the books on the page, and creating buttons to remove books and update their reading status. For more information about the project and the Full Stack JavaScript course, please visit: https://www.theodinproject.com/lessons/node-path-javascript-library.
